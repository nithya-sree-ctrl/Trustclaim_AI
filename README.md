# TrustClaim AI

AI-Powered Multi-Modal Insurance Claim Verification Platform

## Overview

TrustClaim AI is an intelligent insurance claim assessment platform developed for the HackerRank Orchestrate 2026 Hackathon.

The platform uses:

* Gemini Vision
* Computer Vision
* Fraud Detection
* Explainable AI
* Multi-Agent Architecture

to evaluate insurance claims and verify visual evidence.

---

python main.py
